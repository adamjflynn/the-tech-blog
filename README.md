  ## Installation
  
  Required dependancies:
  
  - bcrypt
  - sequelize
  - connect session sequelize
  - express
  - express session
  - express handlebars
  - mysql2
  - dotenv
  
  To run the application locally enter `npm start` in the command line.
  
  ## Usage
  
  Create and account to post your own content and interact with other users posts.
